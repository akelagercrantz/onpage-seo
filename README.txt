OnpageSEO will enable you to set custom meta-data on your pages, such as title, description, keywords and indexing options.

== Description ==

OnpageSEO is a search engine optimization plugin. It aims to provide a complete feature set while small imprint.

It can handle the following meta-data types:

*   title
*   description
*   keywords
*   robots (indexing and follow instructions)

These characteristics can be configured for the following entity-types:

*   Posts
*   Pages
*   Tag archives
*   Author archives
*   Category archives
*   Search results
*   404 page

The main difference to other SEO plugins is the ability to set meta values at three different levels, genrally, the entity type (such as category archives) and the single category.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place the desired meta hooks in your template files.
